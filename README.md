# NHD-240128WG-Example
Example Arduino code for NHD-240128WG displays with RA6963 controller.

Target: `Arduino MEGA`\
Language: `Arduino Sketch`\
Interface: `8080 8-bit Parallel`\
Display Voltage: `5V`\
Logic Voltage: `5V`

# Program Flow
1. Turn on all display pixels.
2. Wait 1 second.
3. Clear the display.
4. Wait 1 second.
5. Display NHD Logo.
6. Wait 1 second.
7. Clear the display.
8. Repeat.
